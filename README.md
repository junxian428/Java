# Java

Java For Linux:

Create Java Program
1. nano filename.java
2. Write Hello World

class filename{
  public static void main(String[] args){
    System.out.println("Hello World!");

  }
}


3. Ctrl X + y and enter to save file
4. Compile

javac filename.java


5. Execute java program

java filename
